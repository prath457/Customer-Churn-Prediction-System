# Customer-Churn-Prediction-System
# 1. Project Overview

# Customer churn prediction helps businesses identify customers who are likely to stop using a service. In this project, we build a machine learning system to predict churn using historical customer data.

Tech Stack

Python

Pandas, NumPy

Matplotlib, Seaborn

Scikit-learn

Business Goal

Identify high-risk customers

Understand why customers churn

Help businesses take preventive actions

# 2. Dataset Description

You can use a telecom churn dataset (common columns):

customerID

gender

SeniorCitizen

tenure

MonthlyCharges

TotalCharges

Contract

PaymentMethod

Churn (Target variable: Yes/No)
